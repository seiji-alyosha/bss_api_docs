Get information about one book.

Gets the title, author, notes, and the added date of a book based on the book id.

Authorizations:
basic_auth
path Parameters
id
required
integer

The books id.

Responses

Production server

https://ennoia.pythonanywhere.com/api/v1/books/{id}
https://username:password@ennoia.pythonanywhere.com/api/v1/books/{id}
